# UPE17

**To add repository**

git init

git remote add origin https://github.com/Noxium/UPE17.git/

**To add files**

git add <filename>

**To push to repo**

git commit -m "Commit message"


**branching**

git checkout -b feature_x
git checkout master
git push origin <branch> // push to branch
git branch -d feature_x // delete branch
git pull // update repo
git merge <branch>
